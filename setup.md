This project uses 3 locally hosted servers to run. There are 3 main directories to be concerned with.
Open a terminal in each of the following directories and run the commands to start their respective servers.

* ~/cs4900-AI-Grader/frontend/
* ~/cs4900-AI-Grader/backend/
* ~/cs4900-AI-Grader/mcp_logic/

## frontend
#### start server
```$ npm start```

---

## backend
#### activate virtual envvironment
```$ source .venv/bin/activate```

#### run server
```$ python manage.py runserver```

---

## mcp_logic
#### activate virtual environment
```$ source .venv/bin/activate```

#### run server
```$ python server.py```

---
